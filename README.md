# TodoList-sequelize
----------------------------
由Express製作的備忘錄
模板引擎使用handlebars
使用sequelize搭配/MySQL資料庫
***
## Installation-安裝說明

1. clone本專案
2. 本專案使用MySQL 請確認安裝
3. npm i 
4. 啟動專案 npm run dev
5. 開啟瀏覽器, 輸入網址 [http://localhost:3000](http://localhost:3000)

***
## Features-專案功能

+ 使用者可以註冊帳號, 登入帳號
+ 使用者可以在todo list 中檢視項目
+ 使用者可以在todo list 中增加項目
+ 使用者可以在todo list 中修改項目
+ 使用者可以在todo list 中刪除項目
